#### 测试样例1

发送正确的用户名和密码，测试样例为

``` json
{
  loan_money:10000
  rate:0.1
  lender_id:1
  debtor_id:1
  product_id:1
  app_date_timestamp:1
  due_date_timestamp:1
  start_date_timestamp:1
  end_date_timestamp:1
  repay_status:ongoing
  order_status:effective
  contract_id:1
}
```

返回的response为：

``` json
{
  "content": {
    "id": 5
  },
  "success": true
}
```

content中包含了该条记录在数据库中的id。
