## 测试内容

测试搜索企业，返回企业名中包含关键词的所有企业的信息。 company_name中可以用空格隔开多个关键词。

#### 测试样例1

测试样例为

``` json
{
  company_name:dio 面包
}
```

返回的response为：

``` json
{
  "content": [
    {
      "address": "埃及",
      "contact": "120",
      "corporate_capital": 13,
      "credit_score": 2.0,
      "description": "你记得自己吃过多少片面包吗",
      "foundation_date": "1996-09-01",
      "id": 3,
      "legal_person_name": "Dio Brando",
      "loan_rate_max": 0.35,
      "loan_rate_min": 0.1,
      "name": "Dio面包坊",
      "register_capital": 13,
      "website": "www.ningenoyameruzo.com"

  ],
  "success": true

```
