## 测试内容

测试添加一条用户评论，包括评论内容和评分。会在LoanProductComment表中增加一条记录，并更新企业用户的信用评分。

#### 测试样例1

测试样例为

``` json
{
  company_name:Dio面包坊
  product_name:Dio面包坊
  user_name:Mike
  comment:wryyyyy
  score:1
}
```

返回的response为：

``` json
{
  "success": true
}
```

success字段为true，代表数据库中的记录已成功被修改。
