## 测试内容

测试获取某个企业的所有产品评价。

#### 测试样例1

测试样例为

``` json
{
  company_name:狼厂
}
```

返回的response为：

``` json
{
  "content": [
    {
      "comment": "balabala_1_1",
      "product_name": "狼厂",
      "score": 1,
      "user_id": 1
    },
    {
      "comment": "balabala_2_1",
      "product_name": "狼厂",
      "score": 4,
      "user_id": 2
    }
  ],
  "success": true
}
```

content中包含了所有对该公司产品的用户评价。
