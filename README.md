#### 开发说明
获取仓库代码
```
git clone https://github.com/Benjamin15122/SoundLoan.git
```
创建各个模块的分支，以demo为例。(已创建分支则可忽略)
```
git checkout -b demo(这里替换成各自的分支名)
```
切换到已创建的分支，以demo为例
```
git checkout demo
```
请在各自的分支上进行开发，并向master分支提交pull request，经过review后方能merge
